# Angular2 RxJS Socket.io chat-application

Features:
- JSON Web Token user authentication.
- Chat functionalities implemented using observable pattern and web socket.

Stack:
- Angular 2.0beta
- RxJS
- Bootstrap
- Socket.io
- Express
